# redux-yep

> 一个基于 `redux` 和 `redux-saga` 的 `react` 应用程序开发状态管理模式！

## 快速开始

****

- `npm install redux-yep -S`
- 或
- `yarn add redux-yep`

## 方法

****

| method       | param     | Description  |
| ------------ | --------- | ------------ |
| `register`   | Imodel    | 注册数据模型 |
| `unregister` | namespace | 卸载数据模型 |
